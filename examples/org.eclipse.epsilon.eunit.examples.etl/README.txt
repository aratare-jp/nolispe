To view the .model files with the Exeed Editor, please register the
two .ecore metamodels in the resources/metamodels folder. To do so,
right click on them and click on "Register EPackages".

To run the JUnit test suite or the EUnit test suites, please use the
included .launch files. You'll need to right click on them and use
the "Run As" -> (launch configuration name) entry of the contextual
menu.
